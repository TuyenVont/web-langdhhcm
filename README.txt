HCMV Blocksy Child V3 Post Template
===================================

Đây là bản child theme cho Blocksy, có:
- Homepage editable trong admin
- Trang bài viết đơn (single post) có layout kiểu editorial giống mockup
- Sidebar phải với khối bài liên quan, bản đồ, CTA hỗ trợ
- Mục lục tự sinh từ heading H2/H3 trong nội dung bài viết

CAI DAT
1. Cài parent theme Blocksy trước.
2. Upload file ZIP child theme này trong Appearance > Themes > Add New > Upload Theme.
3. Activate child theme.
4. Vào Appearance > HCMV Homepage để sửa homepage.
5. Vào Appearance > HCMV Bài viết để sửa layout bài viết đơn.
6. Vào Appearance > Menus để gán menu cho:
   - HCMV Primary Menu
   - HCMV Footer Menu 1
   - HCMV Footer Menu 2
7. Muốn đổi logo: Appearance > Customize > Site Identity.
8. Muốn đổi nội dung bài viết: Posts > Add New / Edit Post.

CO GI CHINH DUOC TRONG ADMIN
- Hero image
- Hero title, description, badge, overlay
- 3 quick links
- Header CTA + placeholder ô search
- Section Bài viết mới nhất (title, desc, số lượng, filter category slug)
- 4 card nhu cầu
- Khối khám phá theo khu vực
- FAQ
- Newsletter
- Footer text + app links
- Danh sách email subscribe xem ngay trong admin page
- Sidebar trang bài viết: recent posts, bản đồ, CTA hỗ trợ, text mục lục/chia sẻ

TRANG BAI VIET DONG THE NAO
- Title, nội dung, excerpt, featured image, category, tag vẫn lấy từ Posts.
- Heading H2/H3 trong bài sẽ tự sinh mục lục.
- Ảnh trong bài chèn bằng block Image bình thường.
- Tag hoặc category sẽ tự hiện ở cuối bài.

GOI Y KHI VIET BAI
- Dùng H2/H3 để có mục lục đẹp.
- Muốn box highlight màu xanh như mockup: thêm class cho Group block là hcmv-note-box.
- Muốn box tóm tắt/lời khuyên: thêm class cho Group block là hcmv-summary-box.
- Trong hcmv-summary-box, nên dùng danh sách bullet để hiện checklist 2 cột trên desktop.

LUU Y
- Homepage lấy bài viết động từ Posts.
- Nếu bài viết không có featured image, theme sẽ tự hiện placeholder ở homepage.
- Nếu bạn muốn thêm nhiều hơn 4 card nhu cầu hoặc hơn 3 FAQ, có thể nâng tiếp lên bản CPT/Repeater.
